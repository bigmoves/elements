Getting Started

Quickslice Elements — Web components for the AT Protocol.

Installation

CDN

<script src="https://unpkg.com/@quickslice/elements"></script>

npm

npm install @quickslice/elements
Quick Start

Add the script and use the components:

<script src="https://unpkg.com/@quickslice/elements"></script>

<qs-actor-autocomplete
  placeholder="Enter a handle"
></qs-actor-autocomplete>

Live Example

Theming

Customize with CSS custom properties:

qs-actor-autocomplete {
  --qs-input-bg: #1a1a1a;
  --qs-input-border: #333;
  --qs-input-border-focus: #00ced1;
  --qs-menu-bg: #222;
  --qs-item-hover: #333;
  --qs-handle-color: #fff;
  --qs-name-color: #888;
}
